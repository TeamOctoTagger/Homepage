Required software

OctoTagger needs the following free software installed in order to run. Download everything either in 64 Bit or 32 Bit.


Python 2
OctoTagger only works with Python 2.7

We used python for the development process, so you have to install the Python SDK at first.

WXPython
Download alternative installer file for Mac OS

For the graphical user interface WXPython helped us a lot. So you have to install this Python library too.

Pillow
Install Pillow via Terminal: pip install Pillow

Now there is just one package left to install. Pillow is responsible for the image rendering.

Pywinlink

Pywinlink was developed by us as a side product of OctoTagger to ensure Windows support. This service makes it possible to generate symlinks in Windows.
