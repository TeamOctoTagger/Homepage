Installation

Besides the regular requirements, for OctoTagger to work in Windows, you need to download and install pywin32. A link is provided below.

pywin32

After all dependencies are satisfied, all you have to do now is to execute the setup.exe of OctoTagger as Administrator. Thats it! OctoTagger is now ready to use, and a shortcut has been created on the desktop.
