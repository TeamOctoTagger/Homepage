Windows

This guide helps you with the installation of OctoTagger on Windows. Please only continue when you have already installed the required software packages.

Before you start to install OctoTagger itself there is one thing left to do. Please download and install the package Pywin32 to continue. Keep in mind that when you decided to download Python, wxPython and Pillow in 64bit you also have to do so with Pywin32, same for 32 bit.

When the download is done, just execute the .exe file.

Pywin32

All you have to do now is to execute the setup.exe of OctoTagger as Administrator. Thats it! OctoTagger is now ready to use, and a shortcut has been created on the desktop.
