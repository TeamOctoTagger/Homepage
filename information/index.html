Have you ever had the need to save the same file in two or more different directories? Unfortunately this is not as easy as it sounds as you have at least one redundant file, which is highly inefficient. We are currently working on a solution for this issue. It's called OctoTagger.

OctoTagger is a simple, but truly powerful software. It allows you to tag files, and it generates a folder for any given tag, in which you can find the relevant files. However, the actual files are located in an invisible directory. So the files in the tag-named directories are just symbolic links pointing at the original files in the hidden directory.

No redundant files - no wasted disc space

A useful extra feature might be the complex search function for tags which includes AND and OR operations. So you can easily find your tagged files within seconds. Furthermore OctoTagger allows you to create advanced output folders with the help of these algorithms instead of just a single tag. Such an advanced output folder contains every file which fits the given algorithm.
